# Детали безопасности

## Защита Windows ИИ

### 1. UAC на максимум
- Панель управления → Учётные записи → Изменить параметры контроля учётных записей → Всегда уведомлять
- Блокирует тихие изменения системы

### 2. Запрет отключения Защитника Windows
- `Win + R` → `gpedit.msc`
- Конфигурация компьютера → Административные шаблоны → Компоненты Windows → Антивирусная программа Microsoft Defender → Выключить антивирусную программу → Отключено
- Или через реестр: `HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender\DisableAntiSpyware = 0`

### 3. Образ диска (резервная копия)
- Установить Macrium Reflect Free
- Создать образ всего диска Windows ИИ
- Сохранить на внешний диск
- Восстановить за 20 минут, если malware сломает систему

### 4. Защита BIOS
- Установить пароль на BIOS (предотвращает изменение порядка загрузки)
- Включить Secure Boot (блокирует загрузку неподписанных систем)
- Отключить USB-boot (предотвращает загрузку с флешки для взлома)

### 5. Сетевая изоляция
- Windows ИИ и Windows Твой никогда не работают одновременно
- BitLocker шифрует Windows Твой, когда он выключен
- Нет сетевого моста между системами

### 6. Политика обновлений
- Автообновления безопасности: ВКЛЮЧЕНЫ
- Функциональные обновления: вручную, раз в 2 месяца
- Образ диска после значительных изменений
