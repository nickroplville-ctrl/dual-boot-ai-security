# Troubleshooting

## Частые ошибки и решения

### 1. Не появляется меню загрузки

**Проблема:** При включении компьютера сразу загружается Windows Твой, без меню выбора.

**Решение:**
1. Загрузитесь в Windows Твой.
2. Нажмите `Win + R` → введите `msconfig` → **Enter**.
3. Перейдите на вкладку **Загрузка**.
4. Проверьте, что обе системы в списке. Если Windows ИИ отсутствует — нажмите **Добавить** и укажите путь к диску Windows ИИ.
5. Нажмите **Применить** → **ОК** → перезагрузите.

---

### 2. BitLocker требует ключ восстановления

**Проблема:** Windows Твой просит 48-значный ключ восстановления вместо пароля.

**Решение:**
1. На другом устройстве (телефон/планшет) зайдите в свой **Microsoft аккаунт**.
2. Перейдите в раздел **Устройства** → найдите свой компьютер.
3. Нажмите **Управление BitLocker ключами** — там будет ключ восстановления.
4. Введите его на компьютере. После загрузки отключите и снова включите BitLocker с паролем.

---

### 3. Hermes не устанавливается в Windows ИИ

**Проблема:** Ошибка при выполнении `curl` или `hermes setup`.

**Решение:**
1. Проверьте установленные программы. В терминале введите:
   - `python --version` (должно быть 3.11+)
   - `node --version` (должно быть 18+)
   - `git --version`
2. Если какая-то команда не работает — переустановите эту программу.
3. Убедитесь, что Windows ИИ подключена к интернету.
4. Если используете WSL для Hermes — убедитесь, что WSL установлен: `wsl --install` в PowerShell от администратора.

---

### 4. Нет свободного места для сжатия диска C

**Проблема:** В Управлении дисками опция "Сжать том" неактивна или показывает мало места.

**Решение:**
1. Очистите диск C:
   - Проводник → Диск C: → правый клик → **Свойства** → **Очистка диска**.
   - Отметьте всё лишнее (Корзина, Временные файлы, Загрузки) → **ОК**.
2. Удалите ненужные программы: **Параметры** → **Приложения** → удалите то, что не используете.
3. Перенесите большие файлы (фото, видео) на внешний диск.
4. Перезагрузите компьютер и попробуйте снова.

---

### 5. Windows ИИ не видит Wi-Fi

**Проблема:** После установки Windows ИИ нет интернета.

**Решение:**
1. Нажмите `Win + X` → **Диспетчер устройств**.
2. Найдите **Сетевые адаптеры**.
3. Если есть жёлтый треугольник — драйвер не установлен.
4. Подключите интернет через кабель (Ethernet) или скачайте драйвер Wi-Fi на другом устройстве и перенесите через флешку.
5. Установите драйвер — обычно с сайта производителя ноутбука/материнской платы.

---

### 6. Rufus не видит флешку

**Проблема:** В Rufus поле "Устройство" пустое.

**Решение:**
1. Убедитесь, что флешка вставлена в USB-порт (лучше USB 2.0, если есть).
2. Попробуйте другой порт.
3. Перезапустите Rufus.
4. Если не помогло — отформатируйте флешку в FAT32 через Проводник, потом снова запустите Rufus.

---

### 7. Забыл пароль BitLocker

**Проблема:** Не помните пароль от Windows Твой, ключ восстановления тоже потерян.

**Решение:**
- К сожалению, данные безвозвратно потеряны. BitLocker не взламывается.
- В будущем: сохраняйте ключ восстановления в надёжном месте (Microsoft аккаунт, распечатка, сейф).

---

### 8. Windows ИИ тормозит

**Проблема:** В Windows ИИ всё медленно, программы долго открываются.

**Решение:**
1. Нажмите `Ctrl + Shift + Esc` → **Диспетчер задач**.
2. Посмотрите, что загружает процессор/память/диск.
3. Закройте лишние программы.
4. Убедитесь, что Windows ИИ установлена на SSD, а не на старый HDD.
5. Если на HDD — рассмотрите перенос на SSD.

---

### 9. Не могу удалить Windows ИИ

**Проблема:** Хотите убрать dual boot, оставить только Windows Твой.

**Решение:**
1. Загрузитесь в Windows Твой.
2. `Win + X` → **Управление дисками**.
3. Найдите раздел Windows ИИ → правый клик → **Удалить том** → **Да**.
4. Появится "Не распределена" область → правый клик на диск C: → **Расширить том** → **Далее** → **Далее** → **Готово**.
5. `Win + R` → `msconfig` → вкладка **Загрузка** → выберите Windows ИИ → **Удалить** → **ОК**.
6. Перезагрузите.

---

### 10. Hermes случайно отключил Защитник Windows

**Проблема:** Защитник Windows выключен, хотя вы настраивали gpedit.

**Решение:**
1. Проверьте, что gpedit настроен правильно (см. security.md).
2. Если Защитник всё равно отключён — восстановите образ диска Windows ИИ из резервной копии.
3. После восстановления проверьте gpedit ещё раз.
