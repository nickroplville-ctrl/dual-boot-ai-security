# Security Checklist / Чек-лист безопасности

[English](#english) | [Русский](#русский)

---

## English

### Check monthly

- [ ] BitLocker is ON on Windows Personal
- [ ] BitLocker password is written and hidden
- [ ] BitLocker recovery key is accessible (Microsoft account / USB / printout)
- [ ] UAC on Windows AI is at maximum
- [ ] Windows AI Defender cannot be disabled (gpedit configured)
- [ ] Windows AI disk image is recent (less than 3 months old)
- [ ] BIOS password is set
- [ ] Secure Boot is ON
- [ ] USB-boot is OFF in BIOS
- [ ] No personal passwords or data in Windows AI
- [ ] Windows AI is updated (last update less than 2 months)
- [ ] Windows AI Defender is running (green checkmark)

### After each AI agent session

- [ ] Did NOT enter BitLocker password in Windows AI
- [ ] Did NOT save passwords in Windows AI browser
- [ ] Did NOT open personal accounts in Windows AI

### If hacked suspected

- [ ] Reboot to Windows Personal
- [ ] Check Windows AI disk image integrity
- [ ] If needed — restore Windows AI from image
- [ ] Change BitLocker password
- [ ] Check BIOS settings for changes

## Русский

### Проверяйте раз в месяц

- [ ] BitLocker включён на Windows Твой
- [ ] Пароль BitLocker записан и спрятан
- [ ] Ключ восстановления BitLocker доступен (Microsoft аккаунт / флешка / распечатка)
- [ ] UAC на Windows ИИ стоит на максимум
- [ ] Защитник Windows ИИ нельзя отключить (gpedit настроен)
- [ ] Образ диска Windows ИИ актуален (сделан менее 3 месяцев назад)
- [ ] Пароль на BIOS установлен
- [ ] Secure Boot включён
- [ ] USB-boot отключён в BIOS
- [ ] В Windows ИИ нет личных паролей и данных
- [ ] Windows ИИ обновлена (последнее обновление менее 2 месяцев)
- [ ] Антивирус Защитника Windows ИИ работает (зелёная галочка)

### После каждой сессии с AI-агентом

- [ ] Не вводил пароль BitLocker в Windows ИИ
- [ ] Не сохранял пароли в браузере Windows ИИ
- [ ] Не открывал личные аккаунты в Windows ИИ

### При подозрении на взлом

- [ ] Перезагрузить в Windows Твой
- [ ] Проверить образ диска Windows ИИ на целостность
- [ ] При необходимости — восстановить Windows ИИ из образа
- [ ] Сменить пароль BitLocker
- [ ] Проверить BIOS на изменения настроек
