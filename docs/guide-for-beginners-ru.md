## Как сделать всё с нуля, если вы ничего не понимаете

---

## Часть 1. Подготовка

### Шаг 1. Проверьте свободное место на диске

1. Откройте **Проводник** (жёлтая папка на панели задач или `Win + E`).
2. Слева нажмите на **Этот компьютер**.
3. Найдите диск **C:**. Под ним написано, сколько свободно.
4. Нужно минимум **400 ГБ свободно**. Если меньше — освободите место (удалите ненужные файлы, программы, очистите Корзину).

### Шаг 2. Скачайте Windows ISO

1. Откройте браузер (Chrome, Edge, Firefox).
2. В адресной строке введите: `microsoft.com/software-download/windows11`
3. Нажмите **Enter**.
4. Прокрутите вниз до раздела **Скачать образ диска Windows 11**.
5. В выпадающем списке выберите **Windows 11**.
6. Нажмите кнопку **Скачать**.
7. Выберите язык **Русский** → **Подтвердить**.
8. Нажмите кнопку **64-bit Скачать**.
9. Файл весит ~5 ГБ. Ждите окончания загрузки. Не закрывайте браузер.

### Шаг 3. Скачайте Rufus

1. В новой вкладке браузера введите: `rufus.ie`
2. Нажмите **Enter**.
3. Прокрутите вниз до раздела **Скачать**.
4. Нажмите на ссылку **Rufus 4.x** (последняя версия).
5. Файл `rufus-4.x.exe` скачается. Не запускайте пока.

### Шаг 4. Подготовьте флешку

1. Возьмите флешку объёмом минимум **8 ГБ**.
2. Всё с неё будет удалено. Скопируйте нужные файлы на компьютер или другую флешку.
3. Вставьте флешку в USB-порт компьютера.

## Часть 2. Запись Windows на флешку

### Шаг 5. Запустите Rufus

1. Откройте папку **Загрузки** (в Проводнике слева).
2. Найдите файл **rufus-4.x.exe**.
3. Дважды щёлкните по нему мышью.
4. Windows может спросить разрешение — нажмите **Да**.
5. Откроется окно Rufus.

### Шаг 6. Настройте Rufus

1. Вверху, в строке **Устройство**, должна быть ваша флешка. Если нет — выберите из списка.
2. Нажмите кнопку **ВЫБРАТЬ** (справа от "Метод загрузки").
3. Откроется окно выбора файла. Найдите скачанный ISO Windows 11 (обычно в папке Загрузки, имя типа `Win11_Russian_x64v1.iso`).
4. Выберите файл → нажмите **Открыть**.
5. В строке **Схема разделов** выберите **GPT** (если у вас современный компьютер, 2015+).
6. В строке **Целевая система** выберите **UEFI (non-CSM)**.
7. Всё остальное оставьте как есть.
8. Нажмите кнопку **СТАРТ** внизу.
9. Появится предупреждение, что флешка будет очищена. Нажмите **ОК**.
10. Ждите. Прогресс-бар зелёный. Когда дойдёт до 100% и надпись **Готов** — можно закрыть Rufus.
11. Флешка готова. Не вынимайте её.

## Часть 3. Сжатие диска C

### Шаг 7. Откройте Управление дисками

1. Нажмите на клавиатуре одновременно **Win + X**.
2. Появится меню в левом нижнем углу. Нажмите на **Управление дисками**.
3. Откроется окно с прямоугольниками — это ваши диски.

### Шаг 8. Сожмите диск C

1. Найдите прямоугольник с надписью **(C:)**. Это ваш основной диск.
2. Щёлкните по нему **правой кнопкой мыши**.
3. В меню выберите **Сжать том...**.
4. Появится окно. В строке **Размер сжимаемого пространства (МБ)** введите: `400000`.
5. Это примерно 400 ГБ. Если хотите другое значение — вводите своё (1 ГБ = 1024 МБ).
6. Нажмите кнопку **Сжать**.
7. Ждите. Может занять 5-15 минут.
8. После сжатия появится чёрная полоса с надписью **Не распределена**. Это ваше свободное место для Windows ИИ.

### Шаг 9. Создайте новый раздел

1. Щёлкните **правой кнопкой мыши** по чёрной полосе "Не распределена".
2. Выберите **Создать простой том...**.
3. Откроется мастер. Нажмите **Далее**.
4. Размер оставьте как есть (весь доступный). Нажмите **Далее**.
5. Букву диска можно оставить по умолчанию (например, D:). Нажмите **Далее**.
6. Файловая система выберите **NTFS**. Нажмите **Далее**.
7. Нажмите **Готово**.
8. Чёрная полоса станет синей с буквой диска. Раздел готов.

## Часть 4. Включение BitLocker

### Шаг 10. Откройте BitLocker

1. Нажмите **Win + S** (поиск).
2. Начните печатать: `bitlocker`.
3. В результатах нажмите на **Управление BitLocker**.
4. Откроется окно Панели управления.

### Шаг 11. Включите BitLocker для диска C

1. Найдите диск **Операционная система (C:)**.
2. Нажмите ссылку **Включить BitLocker**.
3. Появится окно выбора способа сохранения ключа восстановления. Выберите **Сохранить в своей учётной записи Microsoft** (проще всего).
4. Нажмите **Далее**.
5. Выберите **Ввод пароля при запуске**.
6. Придумайте надёжный пароль (минимум 8 символов, буквы + цифры). Запишите его на бумажку и спрячьте.
7. Введите пароль дважды. Нажмите **Далее**.
8. Выберите **Шифровать только занятое место на диске** (быстрее). Нажмите **Далее**.
9. Выберите **Новый режим шифрования** (AES-256). Нажмите **Далее**.
10. Нажмите **Начать шифрование**.
11. Ждите. Может занять от 30 минут до нескольких часов в зависимости от размера диска.
12. После завершения диск C будет защищён. При каждой загрузке Windows Твой будет спрашивать пароль.

## Часть 5. Установка Windows ИИ

### Шаг 12. Перезагрузите компьютер с флешки

1. Флешка с Windows должна быть вставлена.
2. Нажмите **Пуск** → **Питание** → **Перезагрузка**.
3. Сразу после начала перезагрузки начните быстро нажимать клавишу **F12** (или **F2**, **Del**, **Esc** — зависит от производителя компьютера).
4. Появится меню загрузки (Boot Menu) или BIOS.
5. Если Boot Menu — выберите вашу флешку (обычно название типа USB-HDD или Kingston/SanDisk). Нажмите **Enter**.
6. Если BIOS — найдите раздел Boot, поставьте флешку первой, нажмите F10 (Сохранить и выйти).

### Шаг 13. Установите Windows ИИ

1. Появится экран установки Windows. Выберите язык **Русский** → **Далее**.
2. Нажмите **Установить**.
3. Введите ключ продукта (если есть) или нажмите **У меня нет ключа продукта**.
4. Выберите версию Windows (обычно **Windows 11 Домашняя** или **Pro**). Нажмите **Далее**.
5. Примите лицензионное соглашение → **Далее**.
6. Выберите **Выборочная: только установка Windows**.
7. Появится список разделов. Найдите раздел, который вы создали ранее (по размеру ~400 ГБ, неразмеченная область или новый раздел).
8. Выберите его → нажмите **Далее**.
9. Если спросит про форматирование — нажмите **ОК**.
10. Начнётся установка. Ждите 15-30 минут. Компьютер может перезагрузиться несколько раз.

### Шаг 14. Настройте Windows ИИ

1. После установки появится экран настройки.
2. Выберите страну **Россия** → **Да**.
3. Выберите раскладку клавиатуры → **Да**.
4. Пропустите добавление второй раскладки → **Пропустить**.
5. Подключитесь к Wi-Fi (если нужно) или нажмите **У меня нет интернета**.
6. Введите имя компьютера: `Windows-II` (или любое другое).
7. Создайте учётную запись. Имя: `User` (или любое). Пароль: придумайте простой (этот пароль не критичен, тут нет личных данных).
8. Выберите настройки конфиденциальности — отключите всё лишнее (рекламу, отслеживание).
9. Ждите завершения настройки.

## Часть 6. Меню загрузки

### Шаг 15. Проверьте меню загрузки

1. После настройки Windows ИИ перезагрузите компьютер: **Пуск** → **Питание** → **Перезагрузка**.
2. При включении появится чёрный экран с меню:
  - **Windows Твой** (или Windows 11)
  - **Windows ИИ** (или Windows 11, вторая запись)
3. Выбирайте стрелками на клавиатуре → нажмите **Enter**.
4. Если выбрали Windows Твой — введите пароль BitLocker.
5. Если выбрали Windows ИИ — загрузится без пароля.

## Часть 7. Защита Windows ИИ

### Шаг 16. UAC на максимум

1. В Windows ИИ нажмите **Win + S** → начните печатать `uac`.
2. Нажмите **Изменение параметров контроля учётных записей**.
3. Ползунок передвиньте в самый верх — **Всегда уведомлять**.
4. Нажмите **ОК**.
5. Перезагрузите компьютер.

### Шаг 17. Запрет отключения Защитника Windows

1. Нажмите **Win + R**.
2. В окне введите: `gpedit.msc` → нажмите **ОК**.
3. Если Windows скажет, что файл не найден (gpedit.msc отсутствует в Домашней версии), используйте альтернативу:
  - Откройте **Редактор реестра**: `Win + R` → `regedit` → **ОК**.
  - Перейдите: `HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender`
  - Если папки **Windows Defender** нет — создайте: правый клик на **Microsoft** → **Создать** → **Раздел** → назовите `Windows Defender`.
  - Внутри правый клик → **Создать** → **Параметр DWORD (32 бита)** → назовите `DisableAntiSpyware`.
  - Дважды щёлкните → значение `0` → **ОК**.
4. Перезагрузите компьютер.

### Шаг 18. Образ диска (резервная копия)

1. В Windows ИИ откройте браузер.
2. Введите: `macrium.com/reflectfree` → скачайте Macrium Reflect Free.
3. Установите: двойной клик на файл → **Далее** → **Далее** → **Установить**.
4. Запустите программу.
5. Нажмите **Create a backup** → **Image this disk**.
6. Выберите диск с Windows ИИ.
7. Нажмите ссылку под диском **Select a location for your image**.
8. Выберите внешний диск (флешка/внешний HDD) или другой раздел.
9. Нажмите **Next** → **Finish**.
10. Ждите. Образ займёт 20-40 ГБ.
11. Если Windows ИИ сломается — загрузитесь с Macrium Rescue CD (создаётся в программе) и восстановите образ за 20 минут.

## Часть 8. Установка AI-агента

### Шаг 19. Установите необходимые программы

1. **Git**: `git-scm.com` → скачать → установить (всё по умолчанию).
2. **Node.js**: `nodejs.org` → скачать LTS версию → установить.
3. **Python**: `python.org` → Downloads → Windows → Python 3.11+ → установить (галочка **Add Python to PATH**).

### Шаг 20. Установите Hermes

1. Откройте **Терминал** или **PowerShell** (правый клик на Пуск → **Терминал (администратор)**).
  
2. Введите:
  
  ```
  curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
  ```
  
3. Ждите окончания установки.
  
4. Введите: `hermes setup` → следуйте инструкциям.
  
5. Введите: `hermes model` → выберите модель (Claude, GPT-4o, локальную).
  
6. Введите API-ключ (если используете облачную модель).
  

### Шаг 21. Настройте память и скилы

1. Откройте Проводник → папка пользователя → `.hermes`.
2. Создайте файл **SOUL.md** — напишите, кто агент (например: "Ты опытный программист, помогаешь писать код на Python").
3. Создайте файл **USER.md** — напишите о себе ("Я начинающий разработчик, предпочитаю простые решения").
4. Создайте папку **skills** → внутри файл **deploy.md** — опишите процесс деплоя вашего проекта.
5. Для добавления скилов из репозитория: `hermes skills tap add nousresearch/skills`.

### Шаг 22. Установите OpenClaw (альтернатива)

1. В Терминале введите:
  
  ```
  npm install -g openclaw@latest
  ```
  
2. Затем: `openclaw onboard --install-daemon`
  
3. Настройте `openclaw.json` в папке пользователя.
  
4. Для памяти: `openclaw plugins install @mem0/openclaw-mem0`.

## Часть 9. Правила работы

| Что делать | Как делать |
| --- | --- |
| Работать с AI-агентом | Загружайте Windows ИИ |
| Банки, личное, игры | Загружайте Windows Твой |
| Не вводите пароль BitLocker в Windows ИИ | Это откроет доступ к вашим данным |
| Не сохраняйте пароли в браузере Windows ИИ | Там нет ваших личных аккаунтов |
| Обновляйте Windows ИИ | Раз в 2 месяца вручную или автообновления безопасности |
| Делайте образ диска | После настройки — один раз. Потом при major изменениях. |

## Часть 10. Если что-то пошло не так

### Не появляется меню загрузки

- Загрузитесь в Windows Твой → `Win + R` → `msconfig` → вкладка **Загрузка** → проверьте, что обе системы в списке.

### Hermes не устанавливается

- Проверьте, что установлены Python, Node.js, Git. Введите в терминале: `python --version`, `node --version`, `git --version`.

### BitLocker требует ключ восстановления

- Войдите в Microsoft аккаунт на другом устройстве → раздел **Устройства** → найдите свой компьютер → **Управление BitLocker ключами**.

### Windows ИИ сломалась

- Загрузитесь с флешки Macrium Reflect Rescue → выберите образ → восстановите.
